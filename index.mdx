## What is Orange Slice?

Orange Slice is an **AI-powered spreadsheet** that transforms how sales teams research, enrich, and qualify leads. Think of it as a supercharged spreadsheet where every column can be programmable, AI-powered, and connected to hundreds of data sourcesâ€”all without writing complex code.

Unlike traditional spreadsheets, Orange Slice combines:

-  **Familiar spreadsheet interface** with the power of custom JavaScript/TypeScript code
-  **AI chat assistant** that writes code, creates columns, and runs bulk operations for you
-  **Built-in data enrichment** from LinkedIn, company databases, web scraping, and contact APIs
-  **Parallel processing** to enrich thousands of rows automatically with progress tracking

Whether you're finding LinkedIn profiles, generating personalized outreach, or scraping company data, Orange Slice makes it as simple as asking your AI assistant or writing a few lines of code.

## Two Powerful Globals

Every column formula has access to two global objects:

<CardGroup cols={2}>
   <Card title="ctx" icon="code" href="/ctx/overview">
      **Context Object** - Interact with your spreadsheet Read/write cell values, navigate sheets, add rows, and control
      workflow execution.
   </Card>
   <Card title="services" icon="layer-group" href="/services/introduction">
      **Services** - Access data enrichment APIs LinkedIn data, contact info, web scraping, AI generation, and more.
   </Card>
</CardGroup>

```typescript
// Use ctx to read data from your spreadsheet
const name = ctx.thisRow.get("name");
const company = ctx.thisRow.get("company");

// Use services to enrich with external data
const linkedin = await services.person.linkedin.findUrl({
   name: name,
   company: company,
});

// Use ctx to write results back
ctx.thisRow.set({ linkedin_url: linkedin });
```

## Key Capabilities

### Programmable Columns

Every column can execute custom JavaScript/TypeScript code with full AI assistance. Access built-in services for person enrichment, company data, web scraping, AI generation, and more.

```typescript
// Find a person's LinkedIn profile
const linkedin = await services.person.linkedin.findUrl({
   name: ctx.thisRow.get("name"),
   company: ctx.thisRow.get("company"),
});

return linkedin;
```

### AI-Powered Chat Assistant

An intelligent assistant that understands your spreadsheet context and can:

-  Create and modify columns with natural language
-  Write code for enrichment workflows
-  Analyze data and answer questions
-  Run bulk operations across thousands of rows

**Keyboard shortcuts:** `Cmd/Ctrl + J` to toggle chat

### Data Enrichment Library

Integrate with dozens of data providers:

-  **LinkedIn** - Find profiles, enrich data, get employee lists
-  **Contact Info** - Email and phone number discovery
-  **Company Intelligence** - Funding data, employee counts, tech stacks
-  **Web Scraping** - Smart scraping with AI-powered navigation
-  **AI & Research** - Multi-model AI with web-grounded research

### Bulk Processing

Run operations on thousands of rows efficiently with:

-  Parallel processing with automatic rate limiting
-  Dependency resolution (columns run in the right order)
-  Progress tracking and cost monitoring
-  Pause, resume, and retry capabilities
