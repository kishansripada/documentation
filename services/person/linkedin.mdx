## Find LinkedIn URL

Search for a LinkedIn profile URL using various identifying parameters.

### Method

```typescript
services.person.linkedin.findUrl(params)
```

### Parameters

<ParamField path="name" type="string" optional>
  Full name of the person
</ParamField>

<ParamField path="title" type="string" optional>
  Job title or position
</ParamField>

<ParamField path="company" type="string" optional>
  Current or past company name
</ParamField>

<ParamField path="keyword" type="string" optional>
  Additional keywords to refine search
</ParamField>

<ParamField path="email" type="string" optional>
  Email address associated with the profile
</ParamField>

### Returns

Returns a `Promise<string | undefined>` containing the LinkedIn profile URL if found.

### Example

```typescript
const linkedinUrl = await services.person.linkedin.findUrl({
  name: "Sarah Johnson",
  title: "Product Manager",
  company: "Microsoft"
});

console.log(linkedinUrl); // "https://linkedin.com/in/sarahjohnson"
```

## Enrich LinkedIn Profile

Get comprehensive information from a LinkedIn profile including work experience, education, skills, and more.

### Method

```typescript

```

### Example

```typescript
const profile = await services.person.linkedin.enrich({
  url: "https://linkedin.com/in/sarahjohnson"
});

// Access basic information
console.log(`${profile.full_name} - ${profile.headline}`);
console.log(`Location: ${profile.location_city}, ${profile.location_country}`);

// Access work experience
profile.experience?.forEach(job => {
  console.log(`${job.title} at ${job.company_name}`);
  console.log(`Duration: ${job.duration}`);
  console.log(`Description: ${job.description}`);
});

// Access education
profile.education?.forEach(edu => {
  console.log(`${edu.title} - ${edu.major}`);
});

// Access skills
console.log(`Skills: ${profile.skills?.join(", ")}`);
```

## Best Practices

<Tip>
  **Optimize Your Searches**: When using `findUrl`, provide as many parameters as possible to increase accuracy and reduce ambiguous results.
</Tip>

<Warning>
  **Rate Limiting**: LinkedIn enrichment is resource-intensive. Cache results when possible and avoid duplicate requests for the same profile.
</Warning>

<Info>
  **Data Freshness**: The `last_updated` field indicates when the profile was last refreshed. Consider re-enriching profiles that are older than 30 days for critical use cases.
</Info>

## Common Patterns

### Find and Enrich in One Flow

```typescript
async function getFullProfile(name: string, company: string) {
  // First, find the LinkedIn URL
  const url = await services.person.linkedin.findUrl({
    name,
    company
  });
  
  if (!url) {
    throw new Error("Profile not found");
  }
  
  // Then enrich the profile
  const profile = await services.person.linkedin.enrich({ url });
  
  return profile;
}

const profile = await getFullProfile("John Doe", "Acme Corp");
```

### Extract Current Job Information

```typescript
const profile = await services.person.linkedin.enrich({ url });

// Find current position (where date_to is null)
const currentJob = profile.experience?.find(job => job.date_to === null);

if (currentJob) {
  console.log(`Current Role: ${currentJob.title}`);
  console.log(`Company: ${currentJob.company_name}`);
  console.log(`Started: ${currentJob.date_from}`);
}
```

### Calculate Total Experience

```typescript
const profile = await services.person.linkedin.enrich({ url });

console.log(`Total Experience: ${profile.total_experience_duration}`);
console.log(`Total Months: ${profile.total_experience_duration_months}`);
```
