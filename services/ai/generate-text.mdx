## Generate Text

Generate high-quality text content using AI, with optional web search for up-to-date information.

### Method

```typescript
services.ai.generateText(params)
```

### Parameters

<ParamField path="prompt" type="string" required>
  The prompt describing what text to generate
</ParamField>

<ParamField path="model" type="string" required>
  The AI model to use. Options:
  - `"gpt-5-mini"` - Fast and cost-effective
  - `"claude-sonnet-4-5-20250929"` - High quality, balanced performance
</ParamField>

<ParamField path="enableWebSearch" type="boolean" optional default={false}>
  Enable web search to include current information in the response
</ParamField>

### Returns

Returns a `Promise` with:

<ResponseField name="text" type="string" required>
  The generated text content
</ResponseField>

## Examples

### Basic Text Generation

```typescript
const result = await services.ai.generateText({
  prompt: "Write a professional email introducing our new SaaS product to potential customers",
  model: "gpt-5-mini"
});

console.log(result.text);
```

## Use Cases

```typescript
async function generateEmail(context: {
  recipient: string;
  purpose: string;
  tone: string;
}) {
  const result = await services.ai.generateText({

      Purpose: ${context.purpose}
      Tone: ${context.tone}
      Include appropriate greeting and closing.
    `,
    model: "gpt-5-mini"
  });
  
  return result.text;
}

const email = await generateEmail({
  recipient: "potential customer",

  tone: "friendly and professional"
});
```

## Best Practices

<Tip>
  **Detailed Prompts**: The more specific your prompt, the better the output. Include context, tone, length, format, and any specific requirements.
</Tip>

<Warning>
  **Web Search**: Only enable web search when you need current information. It's slower and more expensive than using the model's built-in knowledge.
</Warning>

<Info>
  **Model Selection**: Use `gpt-5-mini` for most tasks (faster and cheaper). Use `claude-sonnet-4-5-20250929` for complex writing requiring nuance and high quality.
</Info>

## Prompt Engineering Tips

### Structure Your Prompts

```typescript
// Good: Structured prompt
const result = await services.ai.generateText({
  prompt: `
    Task: Write a product announcement
    Product: AI-powered analytics tool
    Audience: Data scientists and analysts
    Tone: Excited but professional
    Length: 3 paragraphs
    Include: Key features, benefits, and call-to-action
  `,
  model: "gpt-5-mini"
});

// Less effective: Vague prompt
const result2 = await services.ai.generateText({
  prompt: "Write about our new product",
  model: "gpt-5-mini"
});
```

### Provide Examples

```typescript
const result = await services.ai.generateText({
  prompt: `
    Write a product tagline similar to these examples:
    - "Just Do It" (Nike)
    - "Think Different" (Apple)
    - "The Ultimate Driving Machine" (BMW)
    
    Product: Electric scooter for urban commuters
  `,
  model: "claude-sonnet-4-5-20250929"
});
```

### Specify Format

```typescript
const result = await services.ai.generateText({
  prompt: `
    Create a FAQ section about our pricing.
    Format as:
    Q: [Question]
    A: [Answer]
    
    Include 5 common questions.
  `,
  model: "gpt-5-mini"
});
```

## Error Handling

```typescript
async function generateTextSafely(prompt: string, model: string) {
  try {
    const result = await services.ai.generateText({
      prompt,
      model: model as any
    });
    
    if (!result.text || result.text.length === 0) {
      throw new Error("No text generated");
    }
    
    return result.text;
  } catch (error) {
    console.error("Text generation failed:", error);
    return null;
  }
}
```
