## Generate Text

Generate high-quality text content using AI, with optional web search for up-to-date information.

<Tip>
  **Best for Complex Queries Across Multiple Sources**: Use `enableWebSearch: true` for questions that aren't on a specific website, like "What are the best project management tools?" or "How does blockchain technology work?" This searches across multiple sources to answer your question.
</Tip>

<Warning>
  **Not for Specific Website Data**: If you need to extract information from a specific website, use `services.scrape.website()` + `services.ai.generateObject()` instead. It's more accurate and cost-effective.
</Warning>

### Method

```typescript
services.ai.generateText(params)
```

### Parameters

<ParamField path="prompt" type="string" required>
  The prompt describing what text to generate
</ParamField>

<ParamField path="model" type="string" required>
  The AI model to use. Options:
  - `"gpt-5-mini"` - Fast and cost-effective
  - `"claude-sonnet-4-5-20250929"` - High quality, balanced performance
</ParamField>

<ParamField path="enableWebSearch" type="boolean" optional default={false}>
  Enable web search to include current information in the response
</ParamField>

### Returns

Returns a `Promise` with:

<ResponseField name="text" type="string" required>
  The generated text content
</ResponseField>

## Examples

### Basic Text Generation

```typescript
const result = await services.ai.generateText({
  prompt: "Write a professional email introducing our new SaaS product to potential customers",
  model: "gpt-5-mini"
});

console.log(result.text);
```

### With Web Search

<Info>
  **When to Use Web Search**: Enable web search for questions that require current information from multiple sources, not available on a single specific website.
</Info>

```typescript
const result = await services.ai.generateText({
  prompt: "What are the latest developments in quantum computing as of 2024?",
  model: "claude-sonnet-4-5-20250929",
  enableWebSearch: true
});

console.log(result.text);
// Response will include current information from the web
```

### Answer Questions Across Multiple Sources

```typescript
// Good use case: Question requiring multiple sources
const result = await services.ai.generateText({
  prompt: "What are the top 5 CRM platforms for small businesses and their key differences?",
  model: "claude-sonnet-4-5-20250929",
  enableWebSearch: true

console.log(result.text);

```

## Use Cases

### Question Answering with Web Search

```typescript
async function answerQuestion(question: string) {
  const result = await services.ai.generateText({
    prompt: question,
    model: "claude-sonnet-4-5-20250929",
    enableWebSearch: true
  });
  
  return result.text;
}

// Example: Get current information from multiple sources
const answer = await answerQuestion(
  "What are the latest AI models released in 2024?"
);
```

### Content Summarization

```typescript
async function summarizeWebpage(url: string) {
  const scraped = await services.scrape.website({ url });
  
  const summary = await services.ai.generateText({
    prompt: `Summarize this webpage in 2-3 paragraphs: ${scraped.markdown}`,
    model: "gpt-5-mini"
  });
  
  return summary.text;
}
```

## Best Practices

<Tip>
  **Detailed Prompts**: The more specific your prompt, the better the output. Include context, tone, length, format, and any specific requirements.
</Tip>

<Warning>
  **Web Search**: Only enable web search when you need current information. It's slower and more expensive than using the model's built-in knowledge.
</Warning>

<Info>
  **Model Selection**: Use `gpt-5-mini` for most tasks (faster and cheaper). Use `claude-sonnet-4-5-20250929` for complex writing requiring nuance and high quality.
</Info>
