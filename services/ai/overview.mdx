## Overview

AI services provide powerful tools for generating content, extracting structured data, and conducting deep research using state-of-the-art language models.

## Choosing the Right Tool

<AccordionGroup>
  <Accordion title="ðŸ“Š Extracting Data from a Specific Website" icon="globe">
    **Use: Scrape + Generate Object**
    
    Perfect for gathering structured information from a known website. For example:
    - "How many physicians are at this doctor's office?"
    - "What are the product prices on this page?"
    - "Extract all team members from the about page"
    
    ```typescript
    const scraped = await services.scrape.website({ url: "https://example.com" });
    const data = await services.ai.generateObject({
      prompt: `How many physicians work here? ${scraped.markdown}`,
      schema: z.object({ physician_count: z.number(), names: z.array(z.string()) })
    });
    ```
  </Accordion>

  <Accordion title="ðŸ” Answering Complex Questions (Not on a Specific Site)" icon="magnifying-glass">
    **Use: Generate Text with Web Search**
    
    Best for questions that require searching across multiple sources. For example:
    - "What are the top CRM tools for small businesses?"
    - "How does quantum computing work?"
    - "What are the latest developments in renewable energy?"
    
    ```typescript
    const result = await services.ai.generateText({
      prompt: "What are the best practices for API security in 2024?",
      model: "claude-sonnet-4-5-20250929",
      enableWebSearch: true
    });
    ```
  </Accordion>

  <Accordion title="ðŸ“š Deep, Comprehensive Research" icon="book">
    **Use: Generate Research**
    
    For intensive research requiring thorough analysis across many sources. For example:
    - "Comprehensive market analysis of the EV industry"
    - "Due diligence report on Company X"
    - "Complete competitive analysis of top 5 CRM platforms"
    
    ```typescript
    const result = await services.ai.generateResearch({
      query: "Analyze the current state of the electric vehicle market with growth projections"
    });
    ```
  </Accordion>
</AccordionGroup>

## Common Use Cases

### Extract Structured Data

```typescript
const result = await services.ai.generateObject({
   prompt: "Extract contact information from: John Doe, john@example.com, (555) 123-4567",
   schema: z.object({
      name: z.string(),
      email: z.string(),
      phone: z.string(),
   }),
});

console.log(result.object);
// { name: "John Doe", email: "john@example.com", phone: "(555) 123-4567" }
```

### Generate Content

```typescript
const result = await services.ai.generateText({
   prompt: "Write a professional email introducing our new product",
   model: "gpt-5-mini",
});

console.log(result.text);
```

### Deep Research

```typescript
const result = await services.ai.generateResearch({
   query: "What are the latest trends in AI for 2024?",
});

console.log(result.content); // Comprehensive research report
console.log(`Cost: $${result.cost}`);
```
