## Common Use Cases

### Extract Structured Data

```typescript
const result = await services.ai.generateObject({
   prompt: "Extract contact information from: John Doe, john@example.com, (555) 123-4567",
   schema: z.object({
      name: z.string(),
      email: z.string(),
      phone: z.string(),
   }),
});

console.log(result.object);
// { name: "John Doe", email: "john@example.com", phone: "(555) 123-4567" }
```

### Generate Content

```typescript
const result = await services.ai.generateText({
   prompt: "Write a professional email introducing our new product",
   model: "gpt-5-mini",
});

console.log(result.text);
```

### Deep Research

```typescript
const result = await services.ai.generateResearch({
   query: "What are the latest trends in AI for 2024?",
});

console.log(result.content); // Comprehensive research report
console.log(`Cost: $${result.cost}`);
```
