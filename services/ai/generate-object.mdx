## Generate Structured Object

Extract or generate structured data from natural language using AI. Perfect for parsing unstructured text, form filling, data extraction, and more.

### Method

```typescript
services.ai.generateObject(params)
```

### Parameters

<ParamField path="prompt" type="string" required>
  The prompt describing what object to generate or what data to extract
</ParamField>

<ParamField path="schema" type="any" required>
  The schema defining the structure of the output. Accepts a Zod schema instance or a plain JSON Schema object
</ParamField>

<ParamField path="model" type="string" optional default="gpt-5-mini">
  The AI model to use. Options:
  - `"gpt-5-mini"` - Fast and cost-effective (default)
  - `"gemini-2.0-flash"` - Google's fast model
  - `"gemini-2.5-pro"` - Google's most capable model
  - `"claude-sonnet-4-5-20250929"` - Anthropic's latest model
</ParamField>

### Returns

Returns a `Promise` with:

<ResponseField name="object" type="object" required>
  The generated JSON object matching the provided schema
</ResponseField>

## Examples

### Extract Contact Information

```typescript
const result = await services.ai.generateObject({
  prompt: `
    Extract contact info from this text:
    "Hi, I'm Sarah Johnson. You can reach me at sarah.j@techcorp.com 
    or call me at (555) 123-4567. I work at TechCorp as a Product Manager."
  `,
  schema: z.object({
    name: z.string(),
    email: z.string(),
    phone: z.string(),
    company: z.string(),
    title: z.string()
  })
});

console.log(result.object);
// {
//   name: "Sarah Johnson",
//   email: "sarah.j@techcorp.com",
//   phone: "(555) 123-4567",
//   company: "TechCorp",
//   title: "Product Manager"
// }
```

### Extract Structured Data from Webpage

```typescript
// First scrape the page
const scraped = await services.scrape.website({
  url: "https://example.com/about"
});

// Then extract structured data
const result = await services.ai.generateObject({
  prompt: `Extract company information from this text: ${scraped.markdown}`,
  schema: z.object({
    company_name: z.string(),
    founded_year: z.number().optional(),
    headquarters: z.string().optional(),
    description: z.string(),
    key_people: z.array(z.object({
      name: z.string(),
      title: z.string()
    })).optional()
  })
});

console.log(result.object);
```

## Use Cases

### Parse Resumes

```typescript
async function parseResume(resumeText: string) {
  const result = await services.ai.generateObject({
    prompt: `Parse this resume: ${resumeText}`,
    schema: z.object({
      personal: z.object({
        name: z.string(),
        email: z.string(),
        phone: z.string().optional(),
        location: z.string().optional()
      }),
      summary: z.string().optional(),
      experience: z.array(z.object({
        company: z.string(),
        title: z.string(),
        start_date: z.string(),
        end_date: z.string().optional(),
        description: z.string()
      })),
      education: z.array(z.object({
        institution: z.string(),
        degree: z.string(),
        field: z.string(),
        graduation_year: z.number().optional()
      })),
      skills: z.array(z.string())
    }),
    model: "claude-sonnet-4-5-20250929"
  });
  
  return result.object;
}
```

## Best Practices

<Tip>
  **Clear Prompts**: Be specific in your prompt about what data to extract and from where. Include the source text or context in the prompt.
</Tip>

<Warning>
  **Schema Validation**: The AI will always return data matching your schema, but the quality depends on your prompt. Test with various inputs to ensure reliability.
</Warning>

<Info>
  **Model Selection**: Use `gpt-5-mini` for most tasks (fast and cheap). Use `claude-sonnet-4-5-20250929` or `gemini-2.5-pro` for complex extraction requiring high accuracy.
</Info>

## Advanced Schemas

### Nested Objects

```typescript
const result = await services.ai.generateObject({
  prompt: "Create a company org chart for a tech startup",
  schema: z.object({
    company: z.string(),
    departments: z.array(z.object({
      name: z.string(),
      head: z.string(),
      employees: z.array(z.object({
        name: z.string(),
        title: z.string(),
        reports_to: z.string().optional()
      }))
    }))
  })
});
```

## Error Handling

```typescript
async function extractSafely(text: string, schema: any) {
  try {
    const result = await services.ai.generateObject({
      prompt: `Extract information: ${text}`,
      schema
    });
    
    return { success: true, data: result.object };
  } catch (error) {
    console.error("Extraction failed:", error);
    return { success: false, error: error.message };
  }
}
```
