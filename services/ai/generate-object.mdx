## Generate Structured Object

Extract or generate structured data from natural language using AI. Perfect for parsing unstructured text, form filling, data extraction, and more.

<Tip>
  **Best for Website Data Extraction**: Combine with `services.scrape.website()` to extract structured information from specific webpages. This is the preferred method for answering questions like "How many physicians are at this doctor's office?" or "What are the product specifications on this page?"
</Tip>

### Method

```typescript
services.ai.generateObject(params);
```

### Parameters

<ParamField path="prompt" type="string" required>
   The prompt describing what object to generate or what data to extract
</ParamField>

<ParamField path="schema" type="any" required>
   The schema defining the structure of the output. Accepts a Zod schema instance or a plain JSON Schema object
</ParamField>

<ParamField path="model" type="string" optional default="gpt-5-mini">
   The AI model to use. Options: - `"gpt-5-mini"` - Fast and cost-effective (default) - `"gemini-2.0-flash"` - Google's
   fast model - `"gemini-2.5-pro"` - Google's most capable model - `"claude-sonnet-4-5-20250929"` - Anthropic's latest
   model
</ParamField>

### Returns

Returns a `Promise` with:

<ResponseField name="object" type="object" required>
   The generated JSON object matching the provided schema
</ResponseField>

## Examples

### Extract Contact Information

```typescript
const result = await services.ai.generateObject({
   prompt: `
    Extract contact info from this text:
    "Hi, I'm Sarah Johnson. You can reach me at sarah.j@techcorp.com 
    or call me at (555) 123-4567. I work at TechCorp as a Product Manager."
  `,
   schema: z.object({
      name: z.string(),
      email: z.string(),
      phone: z.string(),
      company: z.string(),
      title: z.string(),
   }),
});

console.log(result.object);
// {
//   name: "Sarah Johnson",
//   email: "sarah.j@techcorp.com",
//   phone: "(555) 123-4567",
//   company: "TechCorp",
//   title: "Product Manager"
// }
```

### Parse Product Information

```typescript
const result = await services.ai.generateObject({
   prompt: `
    Extract product details:
    "The iPhone 15 Pro is available in 128GB, 256GB, and 512GB.
    Prices start at $999. Available in Titanium, Blue, and Natural colors.
    Features include A17 Pro chip, 48MP camera, and USB-C."
  `,
   schema: z.object({
      name: z.string(),
      storage_options: z.array(z.string()),
      starting_price: z.number(),
      colors: z.array(z.string()),
      features: z.array(z.string()),
   }),
});

console.log(result.object);
```

### Extract Structured Data from Webpage

<Info>
  **Recommended Pattern**: This is the preferred approach for extracting specific information from known websites. It's more accurate and cost-effective than using web search for targeted data extraction.
</Info>

```typescript
// First scrape the page
const scraped = await services.scrape.website({
   url: "https://example.com/about",
});

// Then extract structured data
const result = await services.ai.generateObject({
   prompt: `Extract company information from this text: ${scraped.markdown}`,
   schema: z.object({
      company_name: z.string(),
      founded_year: z.number().optional(),
      headquarters: z.string().optional(),
      description: z.string(),
      key_people: z
         .array(
            z.object({
               name: z.string(),
               title: z.string(),
            })
         )
         .optional(),
   }),
});

console.log(result.object);
```

### Count Items on a Webpage

```typescript
// Example: Count physicians at a doctor's office
const scraped = await services.scrape.website({
   url: "https://medicalpractice.com/our-team",
});

const result = await services.ai.generateObject({
   prompt: `How many physicians work at this practice? Extract their names and specialties: ${scraped.markdown}`,
   schema: z.object({
      total_physicians: z.number(),
      physicians: z.array(
         z.object({
            name: z.string(),
            specialty: z.string().optional(),
            credentials: z.string().optional(),
         })
      ),
   }),
});

console.log(`Total physicians: ${result.object.total_physicians}`);
console.log(result.object.physicians);
```
