## Find Company LinkedIn URL

Search for a company's LinkedIn URL using various identifying parameters.

### Method

```typescript
services.company.linkedin.findUrl(params);
```

### Parameters

<ParamField path="name" type="string" optional>
   Company name
</ParamField>

<ParamField path="website" type="string" optional>
   Company website (e.g., "acme.com")
</ParamField>

<ParamField path="location" type="string" optional>
   Company location
</ParamField>

### Returns

Returns a `Promise<string | null>` containing the LinkedIn company URL if found.

### Example

```typescript
const companyUrl = await services.company.linkedin.findUrl({
   name: "Microsoft",
   website: "microsoft.com",
});

console.log(companyUrl); // "https://linkedin.com/company/microsoft"
```

## Enrich Company Profile

Get comprehensive information about a company from LinkedIn. Supports two enrichment levels: **basic** and **extended**.

### Method

```typescript
services.company.linkedin.enrich(params);
```

### Parameters

<ParamField path="id" type="number" optional>
   Company ID
</ParamField>

<ParamField path="shorthand" type="string" optional>
   Company shorthand/slug
</ParamField>

<ParamField path="url" type="string" optional>
   LinkedIn company URL
</ParamField>

<ParamField path="companyDomain" type="string" optional>
   Company domain (e.g., "acme.com")
</ParamField>

<ParamField path="enrichLevel" type="string" default="basic">
   Enrichment level: `"basic"` or `"extended"`
</ParamField>

<Note>Provide at least one of: `id`, `shorthand`, `url`, or `companyDomain`</Note>

### Extended Example

```typescript
const company = await services.company.linkedin.enrich({
   url: "https://linkedin.com/company/microsoft",
   enrichLevel: "extended",
});

// Financial data
console.log(`Revenue: $${company.revenue_annual?.["2024"]?.annual_revenue}`);
console.log(`Stock: ${company.stock_ticker?.[0]?.ticker}`);

// Employee analytics
console.log(`Active Jobs: ${company.active_job_postings_count}`);
console.log(`Employee Review Score: ${company.company_employee_reviews_aggregate_score}`);

// Department breakdown
const deptBreakdown = company.employees_count_breakdown_by_department;
console.log(`Engineering: ${deptBreakdown?.Engineering}`);
console.log(`Sales: ${deptBreakdown?.Sales}`);

// Website traffic
console.log(`Monthly Visits: ${company.total_website_visits_monthly}`);

// Leadership
company.key_executives?.forEach((exec) => {
   console.log(`${exec.member_full_name} - ${exec.member_position_title}`);
});

// Recent news
company.news_articles?.slice(0, 5).forEach((article) => {
   console.log(`${article.headline} - ${article.published_date}`);
});
```

## Use Cases

### Company Research

```typescript
async function researchCompany(companyName: string) {
   // Find the company
   const url = await services.company.linkedin.findUrl({
      name: companyName,
   });

   if (!url) {
      throw new Error("Company not found");
   }

   // Get extended data
   const data = await services.company.linkedin.enrich({
      url,
      enrichLevel: "extended",
   });

   return {
      overview: {
         name: data.company_name,
         industry: data.industry,
         employees: data.employees_count,
         founded: data.founded_year,
      },
      financials: {
         revenue: data.revenue_annual,
         funding: data.last_funding_round_amount_raised,
      },
      culture: {
         reviewScore: data.company_employee_reviews_aggregate_score,
         reviewCount: data.company_employee_reviews_count,
      },
      growth: {
         activeJobs: data.active_job_postings_count,
         websiteTraffic: data.total_website_visits_monthly,
      },
   };
}
```

### Competitive Analysis

```typescript
const company = await services.company.linkedin.enrich({
   url: "https://linkedin.com/company/acme",
   enrichLevel: "extended",
});

// Analyze competitors
company.competitors?.forEach((competitor) => {
   console.log(`${competitor.company_name} - Similarity: ${competitor.similarity_score}%`);
});

// Compare website traffic
company.competitors_websites?.forEach((comp) => {
   console.log(`${comp.website} - ${comp.total_website_visits_monthly} visits/month`);
});
```

### Track Company Growth

```typescript
const company = await services.company.linkedin.enrich({
   url: "https://linkedin.com/company/startup",
   enrichLevel: "extended",
});

// Employee growth over time
company.employees_count_by_month?.forEach((month) => {
   console.log(`${month.date}: ${month.employees_count} employees`);
});

// Job posting trends
company.active_job_postings_count_by_month?.forEach((month) => {
   console.log(`${month.date}: ${month.active_job_postings_count} open positions`);
});
```

## Best Practices

<Tip>
   Use **basic** enrichment for most use cases to optimize performance and costs. Only use **extended** when you need
   detailed financials, analytics, or reviews.
</Tip>

<Warning>
   Extended enrichment is significantly more expensive and slower than basic enrichment. Cache results when possible.
</Warning>

<Info>Company data is updated regularly. Check the `last_updated_at` field to determine data freshness.</Info>
