## Method

```typescript
services.company.getEmployees(params);
```

## Input

<ParamField path="given_company_id" type="number" optional>
   CoreSignal company ID
</ParamField>

<ParamField path="linkedinCompanyUrl" type="string" optional>
   LinkedIn company URL (resolved to company ID automatically)
</ParamField>

<ParamField path="website" type="string" optional>
   Company website (can augment results with website-scraped employees)
</ParamField>

<ParamField path="filters" type="FilterSpec" optional>
   Optional filters to constrain employees (department, management level, location, etc.). Company constraint is applied automatically. See [FilterSpec Reference](/services/filter-spec) for full documentation.
</ParamField>

<ParamField path="options.limit" type="number" optional>
   Max employees per call (500 max). **Start with limit=1 to test.**
</ParamField>

<ParamField path="options.after" type="string" optional>
   Pagination cursor from previous response
</ParamField>

<Note>At least one of `given_company_id`, `linkedinCompanyUrl`, or `website` must be provided.</Note>

## Output

Returns an array of employee profiles (same schema as `person.linkedin.enrich` output).

## Example

```typescript
// Get all employees at a company
const employees = await services.company.getEmployees({
   linkedinCompanyUrl: ctx.thisRow.get("Company LinkedIn"),
   options: { limit: 50 }
});

return employees.length;
```

```typescript
// Get only VPs and Directors in Engineering
const employees = await services.company.getEmployees({
   linkedinCompanyUrl: "https://linkedin.com/company/acme",
   filters: {
      termsAny: [
         { field: "department", values: ["Engineering"] },
         { field: "management_level", values: ["VP", "Director"] }
      ]
   },
   options: { limit: 20 }
});

return employees;
```
